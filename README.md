# Todo List - Vite + React + TypeScript + Tailwind + shadcn/ui

Application Todo List bootstrappee avec:

- Vite + React + TypeScript
- Tailwind CSS
- shadcn/ui (base components + aliases)
- Lucide React (icones)

## Demarrer

```bash
npm install
npm run dev
```

## Build production

```bash
npm run build
npm run preview
```

## Structure principale

- `src/App.tsx`: ecran Todo List (ajout, filtres, toggle, suppression)
- `src/components/ui/*`: composants style shadcn/ui
- `src/lib/utils.ts`: helper `cn`
- `components.json`: configuration shadcn/ui
